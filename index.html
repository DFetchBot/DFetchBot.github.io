FetchBot

FetchBot is a discord bot designed to make moderation easier, as well as providing fun commands for your server members. FetchBot has Auto Moderation, Moderation commands, a leveling system, fun commands, tools for getting information about your server, the members in it and a lot more. FetchBot also has AutoRole, Serverlogs, and welcoming. This is just a few of the features FetchBot provides
